Get started

Install, connect a canvas, and start in under a minute.

Install

npm install @sebastienrousseau/sine-wave-generator

Use

const generator = new SineWaveGenerator({
  el: "#sine",
  maxPixelRatio: 2,
  waves: [{ amplitude: 26, wavelength: 120, speed: 0.8 }]
});

generator.start();

API reference

Minimal surface area, designed to stay predictable.

SineWaveGenerator(options)

Creates a generator bound to a canvas element.

{
  el: HTMLCanvasElement | string,
  waves?: WaveConfig[],
  pixelRatio?: number,
  maxPixelRatio?: number,
  autoResize?: boolean
}

WaveConfig

Configure amplitude, speed, and easing.

{
  amplitude?: number,
  wavelength?: number,
  segmentLength?: number,
  speed?: number,
  phase?: number,
  rotate?: number,
  easing?: (percent, amplitude) => number,
  strokeStyle?: string | null
}

Instance methods

Start/stop and manage waves.

generator.start();
generator.stop();
generator.addWave(config);
generator.removeWave(index);
generator.resize();

Performance

We tune defaults for smoothness, with knobs for heavy scenes.

Cap pixel ratio

Use maxPixelRatio to avoid oversized buffers on high-DPI screens.

Increase segment length

Larger segments reduce per-frame points while preserving visual flow.

Reduce wave count

More waves increases draw cost; balance layering with frame budget.

Important: If you set a high maxPixelRatio, memory use grows quickly on large canvases.
