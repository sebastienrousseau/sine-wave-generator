Quick Start

Browser (script tag)

<canvas id="sine"></canvas>
<script src="https://unpkg.com/@sebastienrousseau/sine-wave-generator"></script>
<script>
  const generator = new SineWaveGenerator({
    el: "#sine",
    waves: [{ amplitude: 26, wavelength: 120, speed: 0.8 }]
  });
  generator.start();
</script>

CommonJS / ESM

const { SineWaveGenerator } = require(
  "@sebastienrousseau/sine-wave-generator"
);

const generator = new SineWaveGenerator({
  el: document.getElementById("sine"),
  maxPixelRatio: 2,
  waves: [{ amplitude: 26, wavelength: 120, speed: 0.8 }]
});

generator.start();

Fundamentals

Core interactions you can remix immediately.

Basic wave

Single wave with default gradient stroke.

const generator = new SineWaveGenerator({ el: canvas });
generator.addWave({
  amplitude: 26,
  wavelength: 140,
  speed: 0.04,
  segmentLength: 10,
});
generator.start();

Layered waves

Stack multiple waves for depth and parallax.

const generator = new SineWaveGenerator({ el: canvas });
generator.addWave({ amplitude: 18, wavelength: 140, speed: 0.04 });
generator.addWave({
  amplitude: 28, wavelength: 200, speed: 0.035,
  strokeStyle: "rgba(14,165,233,0.5)",
});
generator.addWave({
  amplitude: 12, wavelength: 90, speed: 0.05,
  strokeStyle: "rgba(15,23,42,0.35)",
});
generator.start();

Pointer reactive

Pointer movement shapes amplitude and wavelength.

canvas.addEventListener("pointermove", (e) => {
  const rect = canvas.getBoundingClientRect();
  const x = (e.clientX - rect.left) / rect.width;
  const y = (e.clientY - rect.top) / rect.height;
  generator.waves.forEach((wave) => {
    wave.amplitude = 40 + 120 * (1 - Math.abs(y - 0.5) * 2);
    wave.wavelength = 120 + 240 * (1 - Math.abs(x - 0.5) * 2);
  });
});

Dynamic management

Programmatic add/remove of wave layers.

generator.addWave({ amplitude: 12, wavelength: 100, speed: 0.035 });
generator.addWave({ amplitude: 20, wavelength: 140, speed: 0.04 });
// Remove first wave after 2 seconds
setTimeout(() => generator.removeWave(0), 2000);

Performance mode

Low-memory settings with pixel-ratio caps.

const generator = new SineWaveGenerator({
  el: canvas,
  maxPixelRatio: 1,
});
generator.addWave({ amplitude: 18, wavelength: 160, segmentLength: 14 });
generator.addWave({ amplitude: 10, wavelength: 120, segmentLength: 16 });
generator.start();

Custom easing

Swap easing curves for cinematic motion.

const goldenEase = (percent, amplitude) => {
  const g = (1 - 1 / 1.618) / 2;
  if (percent < g || percent > 1 - g) return 0;
  const adj = (percent - g) / (1 - 2 * g);
  return Math.sin(adj * Math.PI) * amplitude;
};
generator.addWave({
  amplitude: 30, wavelength: 160,
  speed: 0.04, easing: goldenEase,
});

Pause and resume

Explicit control over the render loop.

generator.start();
// Pause
generator.stop();
// Resume
generator.start();

API reference

Minimal surface area, designed to stay predictable.

SineWaveGenerator Options

Option Type Default Description
el string | HTMLCanvasElement required Canvas element or CSS selector
waves Wave[] [] Array of wave configuration objects
pixelRatio number devicePixelRatio Device pixel ratio override
maxPixelRatio number 2 Maximum pixel ratio cap for memory/perf
autoResize boolean true Automatically resize canvas on window resize

Wave Options

Option Type Default Description
amplitude number 10 Wave height in pixels
wavelength number 100 Distance between wave peaks in pixels
segmentLength number 10 Rendering segment length (lower = smoother)
phase number random Initial phase offset (radians)
speed number random 0.5-1.0 Animation speed multiplier
easing function Ease.sineInOut Easing function for wave shape
rotate number 0 Rotation angle in degrees
strokeStyle string | null null Custom stroke color

Methods

Method Returns Description
start() this Start the animation loop
stop() this Stop the animation loop
resize() this Manually trigger canvas resize
addWave(config) this Add a wave dynamically
removeWave(index) this Remove wave by index

Easing Functions

Function Description
Ease.sineInOut Smooth sine curve (default)
Ease.sineIn Accelerating sine curve
Ease.sineOut Decelerating sine curve
Ease.linear Linear (no easing)

Performance

Tune defaults for smoothness, with knobs for heavy scenes.

Cap pixel ratio

Use maxPixelRatio to avoid oversized buffers on high-DPI screens.

Increase segment length

Larger segments reduce per-frame points while preserving visual flow.

Reduce wave count

More waves increases draw cost; balance layering with frame budget.

Important: If you set a high maxPixelRatio, memory use grows quickly on large canvases.
